# DojoNum
Développement d’une application web permettant d’apprendre et de réviser les différentes connaissances du judo (quiz général, par grade, vocabulaire, arbitrage, etc...). Suivi de la progression d’un utilisateur (estimation de son grade, des connaissances qu’il lui reste à acquérir, évolution de ses résultats au fil du temps).

Le but de ce projet était majoritairement de me former sur la réalisation d'une application web full stack.
